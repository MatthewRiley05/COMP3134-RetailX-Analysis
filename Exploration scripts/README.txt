HOW TO USE THE DATA EXPLORATION FILES:
-----------
1. Install Python 3, matplotlib, seaborn, NumPy and pandas
2. Specify the path of your CSV files (sales_15.csv, products_15.csv, customers_15.csv) in the script.
3. Run the script

WHAT THE SCRIPTS DO:
---------------------

- 'shopping_mall_analysis.py' - gives you a bar chart of how much sales each shopping mall location generates
- 'age_analysis.py' - analyzes sales patterns across different age groups
- 'gender_analysis.py' - compares sales performance between genders  
- 'category_analysis.py' - examines revenue generation by product categories
