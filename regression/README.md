## Required Packages
Install [pandas](https://pandas.pydata.org/) and [scikit-learn](https://scikit-learn.org/stable/) with pip.


## How to Run

1. **Ensure you have the required data files in the parent directory:**
   - `customers_15.csv`
   - `sales_15.csv`
   - `products_15.csv`

2. **Run the regression script:**
   ```bash
   py regression_main.py
   ```

3. **Output files generated:**
   - `feature_importances_classification.csv` - 1st regression test output file, predicting product id list
   - `regression_coefficients_price.csv` - 2nd regression test output file, predicting price
